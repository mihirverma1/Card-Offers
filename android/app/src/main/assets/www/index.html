Search Sources

Firecrawl Primary crawler. Uses search plus scraped page content when `FIRECRAWL_API_KEY` is configured.
Tavily Optional fallback for search snippets with `TAVILY_API_KEY`.
SerpAPI Optional search fallback through Google results with `SERPAPI_API_KEY`.
Offline rules Local estimates keep the app usable while backend credentials are not deployed.
